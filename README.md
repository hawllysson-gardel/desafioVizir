
## Instruções de Clone

Clone o projeto:

`git clone https://github.com/hawllysson-gardel/desafioVizir.git`

#### Parte 1

Para executar os testes basta acessar o diretório:

`cd desafioVizir`

E executar os testes:

`php .\indexTests.php`

#### Parte 2

Para executar o projeto basta acessar o diretório:

`cd desafioVizir`

E executar o projeto:

`php .\index.php`
